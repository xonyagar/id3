# ID3

**Note:** *⚠️ Under Development ⚠️*

## Usage

### Library
```bash
go get -u github.com/xonyagar/id3
```

### CLI
#### ID3v1
```bash
go install github.com/xonyagar/id3/cmd/id3v1
```
